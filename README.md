# practice_translating_english

A new Flutter project.

## Getting Started

This project is a starting point for a Flutter application.

### Running & Building (Web & Android)

A helper script `run.sh` is provided in the root directory to run or build the application easily.

#### Quick Start (Interactive Menu)
To run the script interactively, execute:
```bash
./run.sh
```

#### Command Line Options
You can also run commands directly:

* **Run on Web (Chrome):**
  ```bash
  ./run.sh web
  ```

* **Run on Android (Device/Emulator):**
  ```bash
  ./run.sh android
  ```

* **Build for Web:**
  ```bash
  ./run.sh build-web
  ```

* **Build Android APK (Release):**
  ```bash
  ./run.sh build-apk
  ```

* **List connected devices:**
  ```bash
  ./run.sh devices
  ```
